# RateLimiter
[![Registry](https://img.shields.io/badge/Registry-SpecWorks-blue)](https://spec-works.github.io/registry/parts/ratelimiter/)


Software component for client-side rate limiting and traffic shaping based on HTTP rate limit headers.

## Specification

Implements [draft-ietf-httpapi-ratelimit-headers](https://datatracker.ietf.org/doc/draft-ietf-httpapi-ratelimit-headers/) - RateLimit Header Fields for HTTP.

See [specs.json](specs.json) for complete specification references.
